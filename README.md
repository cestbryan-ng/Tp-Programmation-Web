## SetUp

- **Laragon** : Version de Laragon utilisée pour la création du projet Laravel [v8.3.0](https://github.com/leokhoa/laragon/releases/download/8.3.0/laragon-wamp.exe)
- **PHP** : [8.3.26 Win32, VS16, x64](https://windows.php.net/downloads/releases/archives/php-8.3.26-Win32-vs16-x64.zip) (probablement déjà installée avec Laragon)
- **Dépendances PHP** : Exécuter `composer install`
- **.env** : Pour créer le .env `cp .env.example .env`
- **Swagger** : `php artisan l5-swagger:generate`
- Pour éxecuter le projet une fois tout installé : `php artisan serve`
