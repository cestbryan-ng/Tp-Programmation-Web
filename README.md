# TP - Programmation Web

Ce dépôt contient les travaux pratiques de programmation web.  
L’organisation est faite par **branches**, afin de séparer l’espace de développement et la version finale de chaque TP.


## Structure des branches

- Chaque TP est divisé en **deux branches** :
  - `tp-dev` : branche de **développement** où le travail est effectué.
  - `tp-main` : branche **finale** correspondant au **rendu officiel** du TP.
